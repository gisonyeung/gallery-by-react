Photo Gallary by React
====================

## Installation

clone the repository to your local computer

```
$ git clone https://github.com/gisonyeung/gallery-by-react.git
```

npm install the dependencies and devDependencies

```
npm install
```

## Start

start the project

```
npm start
```

automatically running on
```
http://localhost:8000/
```
